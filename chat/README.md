# 语音对话程序

基于树莓派的ChatGPT语音对话程序。

## 功能特点

- 语音输入（使用Google Speech Recognition）
- 语音输出（使用pyttsx3）
- ChatGPT对话（使用GPT-4模型）
- 支持中文对话
- 自动环境噪声调整
- 支持唤醒词功能

## 安装步骤

1. 安装依赖包：
```bash
pip install -r requirements.txt
```

2. 配置OpenAI API密钥：
创建`.env`文件并添加以下内容：
```
OPENAI_API_KEY=你的OpenAI API密钥
```

## 使用方法

1. 运行程序：
```bash
python voice_chat.py
```

2. 使用说明：
- 程序启动后，等待"等待唤醒词..."提示
- 说"你好小助手"来唤醒机器人
- 唤醒后，等待"正在听..."提示
- 对着麦克风说话
- 等待AI回复
- 说"退出"、"再见"或"拜拜"结束对话
- 结束对话后，需要重新唤醒才能继续对话

## 注意事项

1. 确保网络连接正常
2. 确保麦克风和扬声器正常工作
3. 需要有效的OpenAI API密钥
4. 唤醒词必须完整说出"你好小助手"

## 可能遇到的问题

1. 麦克风索引问题：
   - 如果系统有多个麦克风，可能需要指定正确的麦克风索引
   - 在代码中修改`sr.Microphone()`为`sr.Microphone(device_index=你的麦克风索引)`

2. 语音识别准确率：
   - 确保环境安静
   - 说话清晰
   - 网络连接稳定
   - 唤醒词识别可能需要多次尝试

3. 语音输出问题：
   - 检查系统音频设置
   - 调整代码中的语速和音量参数
